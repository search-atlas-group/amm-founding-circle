Folio

Book — fixed left TOC + scrollable content
scorecards audits framework refs

Book-style long-form report. Fixed left-anchored TOC (~288px) with a generously-typeset right column. As the reader scrolls, the current section auto-highlights in the TOC and the URL hash updates so any scroll position is link-shareable. Parchment + indigo + gold rule under titles.

Magazine-grade narrative deep-dives. Warm paper ground (#f6f3ec), ink-not-jet-black body, single oxblood accent, serif 19px / 1.7 at a 680px measure. Pull quotes and stat blocks break full-bleed as eye-rest punctuation. Use when the output is meant to be read, not scanned.

Atlas

Dashboard — KPI strip + filterable card grid
leadership pulse OKR roll-ups at-a-glance status

Operational BI dashboard. Top KPI strip + filterable card grid. Designed for "skim in 30 seconds, drill into one card, pivot via chips" leadership snapshots. Sans-serif throughout, tabular numerals, hairline cards, letterspaced uppercase labels. Sparkline + filter chip + status pill are the signature components — and this very page is built from them.

searchable

Catalog

Search-first faceted list with detail drawer
directories registries faceted KBs

Single-surface index for finding one specific record among 60–5000 items. Real ordered list, prominent search with / shortcut, left-rail faceted filters (multi-select OR within, AND across), sort, in-page detail drawer with focus trap, virtual scroll for large datasets. Off-white paper + moss accent + brass-amber active facets.

comparison

Ledger

Side-by-side comparison matrix with scoring
vendor bake-offs A/B reads option tradeoffs

A scoring matrix for N options against M criteria. Real semantic table with sticky row labels + column headers, colorblind-safe semantic tints (good / okay / weak) each with a coloured left rule that double-encodes meaning, a "show big differences" diff toggle, collapsible criterion groups, and a recommendation band at the bottom. Cool slate neutrals, slate-blue accent.

chronological

Timeline

Vertical date spine with event dots + deltas
incident reviews growth narratives release impact

Chronological-narrative report. A vertical date spine runs down the left; each event is an article on the right with a glyph-bearing dot anchored to the spine. Optional per-event metric pills, quote callout, before/after delta block, inline SVG sparkline. Five kinds — incident, release, milestone, anomaly (pulsing), note — colorblind-safe Okabe-Ito palette.
