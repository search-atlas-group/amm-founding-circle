# Correlation worksheet — "who has the ball?"

<!--
  The most valuable step in the whole skill is deciding whether something is
  ACTUALLY still yours to act on, or whether you already handled it somewhere
  else. Run every candidate item through this before it earns a spot in the brief.
  If you can't answer YES to the last question with real evidence, it does NOT
  go in "Do now".
-->

For each thing that surfaced this morning, before you flag it:

- [ ] **Who / what is this really about?**  Person + topic: __________________________

- [ ] **Assemble the whole cluster.** Search that person + topic across a WIDER
      window than one day (try the last ~2 weeks), across ALL your sources —
      inbox, tasks, calendar, CRM. List every place it shows up:
      - source: __________  latest message here (date, from whom): __________
      - source: __________  latest message here (date, from whom): __________
      - source: __________  latest message here (date, from whom): __________

- [ ] **Find the single most recent message across the ENTIRE cluster.**
      Where is it, when, and who sent it? __________________________

- [ ] **Who has the ball right now?**
      - If YOUR last message answered it / diagnosed it / handed it back →
        you owe nothing → this is **CLOSED** or **WAITING ON THEM**.
        Put it in FYI, one line. Do NOT flag it as an action.
      - If the latest message is FROM THEM, dated AFTER your last reply, with a
        real open ask → this is genuinely **YOURS**. It can go in the brief.

- [ ] **Default skeptical.** Still looks open but you're not sure? Assume it's
      already handled in a sibling thread until you find positive evidence
      otherwise. Under-alarm beats crying wolf.

- [ ] **Client isolation.** Everything above stays inside THIS client's context.
      Never pull one client's detail into another client's item or draft.

---

Only items where you can point to a real, most-recent, unanswered ask FROM THEM
belong in "Do now" or "Schedule". Everything else is FYI / already handled —
and that count is the proof your agent did the hard step.
